# Simple Voice AI Client

## Overview

A simple voice AI client that takes uid, channel, and token as params to connect to an Agora channel. Agents can come and go. Useful for testing.

## Features

- Connect to Agora channel with credentials
- Real-time audio streaming
- Simple interface for testing
- Agent connection/disconnection handling

## Prerequisites

- Node.js (v14 or higher recommended)
- Agora Account and API credentials
- Valid channel token

## Getting Started

1. Clone this repository
2. Install dependencies (instructions coming soon)
3. Provide your token, uid, and channel name
4. Run the client

## Parameters

- `uid`: User ID
- `channel`: Channel name
- `token`: Authentication token
