# React JS Voice AI Client

## Overview

React implementation of the Voice AI Client with hooks and component-based architecture. Perfect for modern web applications.

## Features

- React Hooks for state management
- Component-based architecture
- TypeScript support (optional)
- Reusable voice AI components
- Modern React patterns and best practices
- Responsive UI design

## Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn
- React 18+
- Agora Account and API credentials

## Tech Stack

- React 18+
- Agora React SDK
- Modern JavaScript/TypeScript
- CSS Modules or styled-components

## Getting Started

1. Clone this repository
2. Install dependencies: `npm install`
3. Configure your Agora credentials
4. Run development server: `npm start`
5. Build for production: `npm run build`

## Project Structure

```
react-voice-client/
├── src/
│   ├── components/     # React components
│   ├── hooks/         # Custom React hooks
│   ├── services/      # API and Agora services
│   ├── utils/         # Utility functions
│   └── App.js         # Main application
└── package.json
```

## Custom Hooks

- `useAgoraClient`: Manage Agora client connection
- `useVoiceAI`: Handle AI agent interactions
- `useAudioDevice`: Audio device management
