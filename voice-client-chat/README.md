# Voice AI Client with Chat

## Overview

Voice client with integrated text chat alongside real-time audio conversation.
Combines voice and text for a rich conversational experience.

## Features

- Real-time voice conversation with AI
- Integrated text chat interface
- Message history and display
- Synchronized voice and text interactions
- Chat transcription of voice conversation
- Multi-modal communication

## Prerequisites

- Node.js (v14 or higher recommended)
- Agora Account with Voice and RTM enabled
- Backend service for authentication

## Features in Detail

### Voice Communication

- Real-time audio streaming
- Low-latency voice interaction
- AI agent voice responses

### Text Chat

- Send and receive text messages
- Chat history
- Voice-to-text transcription
- Text-to-speech for messages

## Getting Started

1. Clone this repository
2. Install dependencies (instructions coming soon)
3. Configure Agora credentials (both Voice and RTM)
4. Run the client
5. Start chatting via voice and text

## UI Components

- Audio controls
- Chat message list

- Connection status indicator
